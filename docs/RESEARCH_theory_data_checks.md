## 1. 检验一：S 代数时间轴（候选公式 34）——本仓数据不支持，缓立

假说：设备状态分配置面（算子写之间静止）/状态面（自治演化：TTL/定时器/会话老化），
状态面断言须时间索引。预言①sleep 聚集在时变对象附近；②翻转聚集在状态面断言。

- 手册层判据有区分度：10.5 CLI 29 分册 3552 个去重签名中 **202 个（5.7%）** 带时间
  参数（timeout×116/interval×70/expire×23/delay×18/ttl×18）——不是空判据。
- **预言①不可判**：yzg 卷面 13 个去重 sleep 步的 92% 时变邻近富集是**块大小混杂+
  单签名饱和的假象**（`sdns host name [ttl]` 出现在 79% 卷面基线块；按块大小分层后
  富集归零）；且 13 个 sleep 全是「等配置生效」型，无一 TTL/定时器等待型——yzg
  （sdns listener 配置域）没有给该预言施展的对象。
- **预言②反对多于支持**：8 个翻转实例中 2 例翻在 `show sdns listener` **配置面断言**
  （直接反例）；翻转组断言目标的「探测占比」（81%）与稳定 fail 组（78%）无差；
  引擎归因 0 例指向自治演化，主导解释是床污染（run7-9 恰为污染期）。
- **附带理论修正**：「配置面在算子写之间静止」的「算子」须限定为**本案算子**——
  共享床是多写者，他案写造成的「配置面翻转」归 infra T1 域，不是时间轴反例的
  同时也不是时间轴的辖区。
- 唯一强定性支持：zhaiyq 22/50 时序翻转（persistence TTL 域、dig 查询、子集 pass
  整卷漂移）——方向完全一致但**原始卷不可得**（仅 memory 记载）。
- **裁决：缓立**。域外背书强（见 §5 主题4）、域内数据待补；前置=一个 persistence/
  health-check 域批次的原始数据。手册时变签名表可先落文法层数据（零代码增长）。
  sleep 治理维持队列末位。

## 3. 检验三：校准信号存在性——有信号，限定用途

- 逐案配对 n=72（dongkl 33 + selfheal1 13 + yzg 26），标签=盘上可还原的最早 R1。
- **意图复杂度特征批内判别力成立**（dongkl 批内 n=33）：字数>99 → R1 25% vs 82%
  （-57pp，Fisher p=0.0016）；步数/检查点数/跨设备同向（p=0.005-0.013）。特征
  logistic LOO AUC=**0.788**。
- **K 覆盖词面特征证伪**：footprint/先例词面命中与复杂度共线（Spearman 0.52-0.63）
  且方向为负——词面近似没测到 K；测真 K 覆盖需历史 K 快照+语义检索（构造路径
  证伪，假设未判）。
- 与中心论断相容：批间 R1（38%→55%→92%）由 K 状态主导；意图特征是**批内残余
  真实信号**——K 定批级、复杂度定案级排序，两层各管各的。
- **裁决：立（限定用途）**——预测器决定资源（思考深度/先例注入量/k 采样数/ask
  提前），oracle 决定交付；**永不当质量门**（AUC 0.7-0.8 是文献路由器天花板）。

## 4. 文献调研一：生成质量定量线——翻案线是活路

- **我们证伪的，文献同步证伪**：言语自评置信判别力弱（Xiong ICLR'24 2306.13063、
  Spiess ICSE'25 2402.02047）；942 对删 grade 与文献并轨。翻案构件（特征路由/
  稳定性）在同批横评里显著优于自评——**不同谱系**。
- 难度预测/路由成熟（RouteLLM 2406.18665、BEST-Route ICML'25 2506.22716、
  IRT-Router ACL'25 2506.01048）；用法铁律=预算分配非质量门。
- **旧碰撞熵理论获 Nature 级同构背书**：semantic entropy（Nature 630:625, 2024；
  碰撞熵=Rényi-2）；代码域正确移植=CodeT 式执行等价（2207.10397）——对我们即
  断言集/步骤结构等价类，structural_gate 现成可算。**HACK 警告**（2510.24222）：
  模型稳定地一致地错时熵为零——稳定性必要非充分，oracle 地位不动。
- 测试生成域**无先验质量预测先例**（空白非证伪）；我们现行「机械门+oracle+修复
  环」与 Meta Assured LLMSE（TestGen-LLM 2402.09171/ACH 2501.12862）精确同构。
- **最优组合=文献级联范式**：便宜难度预测全量跑 → 高危子集 k 采样稳定性检测 →
  机械门+上机 oracle 不变。
