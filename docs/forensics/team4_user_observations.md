**触发上报的三类问题（用户原话标准）**：
- **① user 看不懂**：黑话/内部代号/无解释的 autoid 编号、user-facing 题面里的英文、格式崩坏/乱码、表格塞不下。
- **② user 不知道怎么回答**：ask/问询面板上题面或选项给的上下文不足以做选择——选项含混、缺「为什么问我」、选项预设了屏幕从没展示过的知识、没有明显的安全默认项。
- **③ 界面 error**：traceback、error 字样、pane 崩溃、渲染乱码，或活跃批次期间进度连续 2 次抓屏卡死不动。

**纪律**：只读观察，绝不向 surface:18 发键/发文字；只凭屏幕可见内容判断；证据必须逐字摘录不转述；触发即 SendMessage 上报 team-lead（leader 单点路由，不直接找 Test-Eng）；无事静默。
